# Budget SMS Assistant

Budget SMS Assistant is a text-message based budgeting tool that allows users to set budgets and track spending by sending SMS commands.

## How It Works

Users interact with the service by texting a phone number.

Example commands:

- "budget groceries 400"
- "spent 45 groceries"
- "groceries balance"

The system automatically processes the request and replies with updated budget totals.

## Technology

- Twilio (SMS messaging)
- Make.com (automation workflows)
- Airtable (data storage)
- OpenAI (natural language processing)

## Example Interaction

User: budget groceries 400

Bot: Grocery budget set to $400.

User: spent 45 groceries

Bot: Groceries spending:
$45 / $400
Remaining: $355
